# Customer Prediction

## Project Overview
This project aims to predict clients who are likely to become customers of an online learning platform ExtraaLearn.

## Dataset
The dataset includes customer information, like age, profession, and interaction type with service, and a binary variable depending on whether they became customers or not.

## Methodology
- Data Cleaning and Preprocessing
- Model Training and Evaluation using:
  - Logistic Regression
  - Decision Trees
  - Random Forest

## Results
- Achieved an accuracy of 82% and positive recall of 85% with a tuned Random Forest Classifier.
- Key features influencing default prediction were identified.

## Files
- `SC_Customer_Prediction.ipynb`: Jupyter Notebook containing the analysis and model implementation.
- `ExtraaLearn.csv`: Dataset used for the project.
